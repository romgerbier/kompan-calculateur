# KOMPAN Calculateur — by ROMGER

Application web mobile pour chantiers d'aires de jeux KOMPAN.

## Installation sur iPhone
1. Ouvrir dans Safari
2. Partager → Sur l'écran d'accueil
3. Logo KOMPAN apparaît

## Fonctionnalités
- 12 calculateurs terrain
- Normes EN 1176, EN 16630
- Schémas KOMPAN officiels
- Fonctionne hors ligne
